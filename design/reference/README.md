# Visual Reference Mockups

Canonical visual reference mockups for the Project Vintage UI. Two files in this directory:
- initial-report.html — the gated initial-report view (signed-in free tier)
- full-report.html — the unlocked full-report view (Pro tier)

Open in a browser to view. The HTML contains the exact layout, classes, spacing, and hex codes used in the canonical design. When implementing UI, reference both DESIGN.md (the written system) and these HTML files (the visual implementation) together.
